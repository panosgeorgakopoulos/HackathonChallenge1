## üéØ The Challenge

Marine heatwaves are prolonged periods of anomalously high sea surface temperatures. For the aquaculture industry, these events can be catastrophic, leading to:
* High mortality rates in fish stocks.
* Significant economic losses for producers.
* Long-term environmental stress.

**The Goal:** Create a reliable tool that analyzes historical and real-time data to predict the onset of these heatwaves.
